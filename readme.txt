== Description ==

AWP antivirus scans all of your local files to find malware code or file injection. It uses an external database to compare the potentially dangerous code found in the files. If an unknown piece of code is found, it is submitted to the database for human validation.

Upload the AWP Antivirus plugin to your blog, Activate it, then enter your [API key](http://www.awp-antivirus.com/).

1, 2, 3: You're done!

== Frequently Asked Questions ==

= How is malware code found ? =

Every suspicious code is sent to a central database to be compared. If the code is unknown, it will be checked by human intervention.

= Will my files be deleted ? =

If malware code is found, files will either be deleted or edited if you have entered a valid API key for your website. If it is a new file that was created by the virus, it will be deleted, if it is only a piece of code that was injected in an existing file, the piece of code will be removed.

= What is the firewall service ? =

It is a list of htaccess rules that protect your website from being hacked.

= How do I get an API key ? =

You can buy it via our [website](http://www.awp-antivirus.com/), you will receive it within a few minutes.
